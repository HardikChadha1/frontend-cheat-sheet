HTML

Tags

  • <h1> - Defines the largest heading
  • <p> - Defines a paragraph
  • <a> - Defines a hyperlink
  • <div> - Defines a division or section
  • <span> - Defines an inline section
  • <img> - Embeds an image
  • <ul> - Defines an unordered list
  • <ol> - Defines an ordered list
  • <li> - Defines a list item
  • <table> - Defines a table
  • <tr> - Defines a table row
  • <td> - Defines a table cell
  • <th> - Defines a table header
  • <form> - Defines a form
  • <input> - Defines an input field
  • <button> - Defines a clickable button
  • <label> - Defines a label for an input element
  • <textarea> - Defines a multiline text input
  • <select> - Defines a dropdown list
  • <option> - Defines an option in a dropdown list

Attributes

  • href - Specifies the URL for a hyperlink
  • src - Specifies the URL for an image
  • alt - Provides alternative text for an image
  • id - Specifies a unique id for an element
  • class - Specifies one or more class names for an element
  • style - Specifies an inline CSS style for an element
  • title - Provides extra information about an element (displayed as a tooltip)
  • placeholder - Specifies a short hint in an input field
  • value - Specifies the initial value of an input element
  • name - Specifies the name of an input element (used to reference form data after form submission)
  • rows - Specifies the number of visible text lines in a textarea
  • cols - Specifies the visible width of a textarea
  • for - Specifies which form element a label is bound to
  • selected - Specifies that an option should be pre-selected when the page loads
  • checked - Specifies that an input element should be pre-checked when the page loads
  • disabled - Specifies that an input element should be disabled

Syntax

HTML uses opening and closing tags to structure content:

<tagname>Content goes here...</tagname>
